# Abhishek Bagde — Personal Portfolio

Welcome to the source code for my personal portfolio website: [abhishek100100.github.io](https://abhishek100100.github.io/).

## 🚀 Overview
This repository hosts a high-performance, minimalist personal website showcasing my trajectory as a System Software Engineer. The design focuses on elegance, readability, and a premium tech-forward aesthetic, built entirely from scratch without heavy frameworks.

## 🎨 Design & Aesthetics
- **Premium Typography**: Utilizes a blend of `Caveat` for an organic signature, `Plus Jakarta Sans` for clean UI reading, and `Newsreader` for editorial emphasis.
- **Dynamic Themes**: Fully responsive Light & Dark modes with smooth transitions. The dark mode features deep slate backgrounds to reduce eye strain while maintaining perfect contrast.
- **Micro-Interactions**: Features elegant scroll reveals using `IntersectionObserver`, subtle drop-shadow hover states, and dynamic gradient-text animations on key phrases.
- **Asset Optimization**: Smart CSS filters are used to ensure high-contrast imagery (like blueprint-inversions for abstract art and logo backings) across both themes.

## 🛠️ Tech Stack
- **HTML5**: Semantic layout and robust structure.
- **CSS3**: Vanilla CSS utilizing modern custom properties (CSS variables) for theme switching, flexbox/grid layouts, and smooth animations.
- **JavaScript (Vanilla)**: Lightweight DOM manipulation for theme toggling (with `localStorage` persistence), smooth anchor scrolling, and scroll-reveal observers.
- **Deployment**: Hosted statically and continuously deployed via **GitHub Pages**.

## 💻 Running Locally
Since this is a static site with no build-steps or package dependencies, running it locally is instantaneous:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Abhishek100100/abhishek100100.github.io.git
   cd abhishek100100.github.io
   ```

2. **Serve the directory**:
   You can use any local static web server. For example, using Python 3:
   ```bash
   python -m http.server 8000
   ```

3. **View**:
   Open `http://localhost:8000` in your web browser.
